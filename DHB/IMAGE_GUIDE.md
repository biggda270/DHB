## üí° DESIGN TIPS FOR BEST RESULTS

1. **Keep images dark** - Use darker images or add dark overlays for text readability
2. **Consistent style** - Choose images with similar color tones
3. **High resolution** - Use at least 1920px wide for hero images
4. **Optimize file size** - Compress images to under 500KB each
5. **Attribution** - If using copyrighted images, add credits in footer
